My Projects

 

Here’s a bit of information about the projects I’ve stored on Github.

 

umpire: This is a web-based tool for analysing data for a local football league’s umpires. They use it to see how many games umpires have officiated with other umpires, for each club and team, and so on. It helps them plan their upcoming assignments.
